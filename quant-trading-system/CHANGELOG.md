# 更新日志 (Changelog)

## [v1.2.0] - 2026-04-12

### 重大功能发布 - 核心系统完成

本次版本完成了量化交易系统的全部核心功能（Phase 1-6），系统已具备从策略研发到实盘交易的完整能力。

#### Phase 6: 智能分析系统 ✅
- 基本面分析引擎 (695行) - 估值/成长/盈利/财务健康评分
- 智能选股器 (477行) - 多条件组合筛选 + 4种预设策略
- 多因子评分模型 (488行) - 价值/成长/质量/平衡模型
- 智能分析API - `/fundamental`, `/screen`, `/multifactor`
- 选股器前端重构 - 连接真实API数据

#### Phase 5: 风控系统完善 ✅
- 风控管理API (15,786 bytes) - 13个REST端点 + WebSocket
- 7种风控规则：仓位/止损/熔断/频率/连续亏损/涨跌停/黑名单
- 实时风险评分 (0-100) 仪表盘
- 紧急清仓功能（二次确认保护）
- 风险预警通知系统

#### Phase 4: 实盘监控系统 ✅
- 信号发布器 - WebSocket实时推送，支持去重和节流
- 自动交易器 - 4种模式：AUTO/MANUAL/SIMULATE/PAUSE
- 实盘内阁 - 统一管理和协调
- 实盘监控视图 - 实时信号、交易记录、绩效展示

#### Phase 3: 策略管理 ✅
- 策略CRUD、版本管理、参数优化
- 网格搜索 + 遗传算法优化器
- 策略编辑器前端

#### Phase 2: 回测系统 ✅
- 回测引擎 - 事件驱动、滑点、手续费
- 回测指标 - 夏普比率、最大回撤、年化收益
- 回测任务管理

#### Phase 1-1.5: 核心交易闭环 ✅
- 三省六部架构 - 数据/信号/风控/执行分离
- 订单状态机 - 完整生命周期管理
- 数据质量监控
