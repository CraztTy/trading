# 金融数据源替代方案对比

## 概述

AKShare虽然免费，但存在稳定性问题（基于爬虫，源站改版会失效）。以下是各种替代方案，从免费到付费，从简单到专业。

---

## 一、免费开源方案

### 1. Baostock (推荐)

**简介**: 证券宝，免费的金融数据平台

**优点**:
- ✅ 完全免费，无需注册
- ✅ 数据稳定，官方维护
- ✅ A股历史数据完整（2000年至今）
- ✅ 支持复权数据
- ✅ 财务数据、季频数据

**缺点**:
- ❌ 实时数据有延迟（15分钟）
- ❌ 数据种类不如AKShare丰富

**适用场景**: 回测、历史数据分析

**接入示例**:
```python
import baostock as bs

# 登录（可匿名）
bs.login()

# 获取历史K线
rs = bs.query_history_k_data_plus(
    "sh.600519",
    "date,code,open,high,low,close,volume",
    start_date='2024-01-01',
    end_date='2024-12-31',
    frequency="d",
    adjustflag="3"  # 复权类型
)

# 获取财务数据
rs = bs.query_profit_data(code="sh.600519", year=2024, quarter=4)
```

### 2. Tushare (免费版)

**简介**: 国内最知名的金融数据接口

**官网**: https://tushare.pro

**优点**:
- ✅ 数据质量高，维护活跃
- ✅ 数据种类丰富
- ✅ 社区活跃，文档完善
- ✅ 支持多种编程语言

**缺点**:
- ❌ 免费版有积分限制（120积分/分钟）
- ❌ 高级数据需要付费
- ❌ 需要注册获取Token

**免费版限制**:
| 接口 | 限制 |
|------|------|
| 日线数据 | 5000条/次 |
| 分钟数据 | 限量 |
| 财务数据 | 有限制 |
| 实时行情 | 不可用 |

**适用场景**: 中小规模数据需求、学习研究

**接入示例**:
```python
import tushare as ts

# 需要注册获取token
pro = ts.pro_api('your_token_here')

# 获取日线数据
df = pro.daily(ts_code='000001.SZ', start_date='20240101', end_date='20241231')

# 获取分钟数据（需积分）
df = pro.minute(ts_code='000001.SZ', freq='1min')
```

### 3. Yahoo Finance (yfinance)

**简介**: Yahoo Finance 数据接口

**优点**:
- ✅ 全球股市数据（美股、港股、A股）
- ✅ 完全免费，无需注册
- ✅ Python库简单易用
- ✅ 历史数据完整

**缺点**:
- ❌ A股数据有时不准确
- ❌ 实时数据延迟
- ❌ 可能不稳定（API会变动）

**适用场景**: 美股数据、全球市场分析

**接入示例**:
```python
import yfinance as yf

# A股需要加后缀
# .SS = 上海, .SZ = 深圳
stock = yf.Ticker("600519.SS")

# 获取历史数据
hist = stock.history(period="1y")

# 获取实时数据
info = stock.info
```

## 八、接入建议

### 1. 多源备份策略
```python
class DataProvider:
    def __init__(self):
        self.primary = AKShareGateway()
        self.backup1 = BaostockGateway()
        self.backup2 = TushareGateway()
    
    async def get_tick(self, symbol):
        # 优先主源
        try:
            return await self.primary.get_tick(symbol)
        except:
            # 降级到备用源
            try:
                return await self.backup1.get_tick(symbol)
            except:
                return await self.backup2.get_tick(symbol)
```

### 2. 缓存策略
- 实时行情: 3-5秒缓存
- 日K线: 1天缓存
- 基础信息: 7天缓存
- 财务数据: 永久缓存（季度更新）

### 3. 降级方案
1. AKShare失败 → Baostock
2. Baostock失败 → Tushare
3. 全部失败 → 模拟数据（Mock）
