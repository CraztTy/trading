## Tushare 积分规则

### 积分获取方式

| 方式 | 积分 | 说明 |
|------|------|------|
| 注册赠送 | 120 | 新用户默认积分 |
| 完善信息 | 20 | 绑定手机号、邮箱 |
| 实名认证 | 20 | 完成实名认证 |
| 邀请好友 | 50 | 每邀请一个好友 |
| 充值 | - | 1元 = 10积分 |

### 接口权限要求

| 接口 | 所需积分 | 每分钟限制 |
|------|---------|-----------|
| `stock_basic` (股票列表) | 200 | 500次 |
| `daily` (日线) | 200 | 500次 |
| `weekly` (周线) | 200 | 500次 |
| `monthly` (月线) | 200 | 500次 |
| `daily_basic` (每日指标) | 600 | 800次 |
| `moneyflow` (资金流向) | 2000 | 500次 |
| `limit_list` (龙虎榜) | 2000 | 500次 |

### 您的当前状态

## 解决方案

### 方案1: 获取积分（免费）

1. **完善个人资料** (+40积分)
   - 绑定手机号: +20
   - 绑定邮箱: +20
   - 实名认证: +20
   
2. **邀请好友** (+50积分/人)
   - 每邀请一个注册好友得50积分
   - 邀请2-3人即可获得足够积分

3. **完成新手任务** (+60积分)
   - 关注公众号
   - 加入社区

**总计可获得**: 120 + 40 + 60 = 220+ 积分

完成以上步骤后，即可访问基础数据接口。

---

### 方案2: 充值积分（最快）

充值 20 元 = 200 积分，立即解锁基础接口。

---

### 方案3: 使用替代数据源

如果暂时无法获取积分，系统已自动降级到 **Baostock** 或 **模拟数据**。
