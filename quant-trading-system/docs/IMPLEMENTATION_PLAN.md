## 数据库 + 模拟盘 验收标准

1. **下单流程**
   - 能成功创建限价单
   - 资金正确冻结
   - 订单状态正确流转

2. **撮合成交**
   - 限价单按价格优先撮合
   - 大单部分成交
   - 成交后资金/持仓正确更新

3. **数据一致性**
   - 连续运行7天无对账差异
   - 重启后持仓/资金状态正确
   - 资金流水总和等于账户余额变动

4. **T+1结算**
   - 当日买入不可卖出
   - 日终后变为可卖
